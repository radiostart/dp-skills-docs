- [설치 및 초기 세팅](installation.md) — 플러그인 등록, 팀 workspace 부트스트랩, 환경변수
- [Quick Start](quick-start.md) — 5분. `team-init → project → @ag-planner` 한 사이클을 명령 그대로 따라 합니다.
- [Deep Walkthrough](getting-started.md) — 기획서 fetch 부터 PR 까지 전체 흐름을 한 번 끝까지 따라갑니다.
